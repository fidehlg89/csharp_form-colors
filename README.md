# PruebaColores

Este es un proyecto de aplicación de escritorio (.NET WinForms) escrito en Visual Basic.
La aplicación principal cuenta con un menú que permite abrir tres formularios distintos interactivos:
- Formulario de Colores
- Formulario de Formas
- Formulario de Formas y Colores (Ambos)

## Tecnologías
- **Lenguaje:** Visual Basic .NET (`.vb`)
- **Tipo de Proyecto:** Windows Forms Application (WinForms)

## Requisitos
- .NET Framework
- Visual Studio (para abrir y compilar la solución `PruebaColores.sln`)

## Cómo ejecutar
1. Clona el repositorio.
2. Abre la solución `PruebaColores.sln` con Visual Studio.
3. Compila el proyecto (Ctrl + Shift + B).
4. Inicia la aplicación (F5).
